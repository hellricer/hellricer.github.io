URxvt extensions

[April 16, 2020]

URxvt was my "weapon of choice" for many years. For me, it always constituted perfect blend between blazing-fast and feature-full terminal emulator.

It might seem basic at first, but the powerful urxvtperl(3) interpreter allows users to extend it with pretty much any feature of other terminal emulators like gnome-terminal or iTerm2 and more.

This post contains a curated set of some of these extensions with small description of what they do.

Changing colors & font-size on the fly.

This is very useful when you're screen-sharing your hidpi desktop and people complain they can't read anything, or when logs get too long and you want to see them without line-breaks, etc.

  • https://github.com/majutsushi/urxvt-font-size (included in official packages)
  • https://github.com/felixr/urxvt-theme-switch

You can also change whole font family with https://github.com/jmccarv/urxvt-fontcycle:

Open new window in current directory

Pretty much standard function of most terminal emulators, which can save you a ton of keystrokes, can be enabled with this extension.

  • https://github.com/effigies/urxvt-perl/blob/master/newterm

Overlay windows

Maybe you noticed that when you press Ctrl+Shift, a pop-up window appears in the corner. URxvt provides an API for creating pop-ups like that.

The official extension overlay-osc provides custom OSC for general purposes and many other extensions use it for displaying some info.

  • resize-overlay from perlguy displays pop-up with number of columns and rows when resizing.
  • official extension digital-clock shows time in top-right window (although I fail to see how that could be useful :P)

Status line

If you don't feel like using tmux but still would like to have some status line in your terminal, you can use one of these:

  • https://github.com/su8/pinky-urxvt
  • https://github.com/ryuslash/urxvt-modeline
  • https://github.com/Jeansen/cdmn

Auto-completion

  • https://github.com/Vifon/autocomplete-ALL-the-things
  • https://github.com/stdliu/urxvt-complete

Mouseless copying

  • https://github.com/bartman/urxvt-scripts/blob/master/mark-and-yank
  • https://github.com/quite/urlpicker
  • https://github.com/Kuroneer/urxvt-buffer2vim

Desktop notifications

  • lpenz/urxvt-ext-notify-osc
  • https://github.com/Waldteufel/osc_notify

Clearing the screen

  • https://github.com/burton-aad/urxvt-softclear/blob/master/soft-clear
  • https://github.com/Maelan/urxvt-plugins/blob/master/clear

Miscelaneous, development

  • https://github.com/verrens/urxvt-pcon

    • provides Perl interpreter console.

  • https://github.com/kovetskiy/urxvt-fifo

    • makes it possible to send keystrokes to running instance through a pipe.

  • https://github.com/Roliga/urxvt-xresources-256

    • Many Xresource themes define more than sixteen colors, but urxvt load only first sixteen. This extension makes it possible to load all of them.
    • note: you might want to compile urxvt from sources with truecolor support.
